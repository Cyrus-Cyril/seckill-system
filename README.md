# 商品库存与秒杀系统

## 项目简介

本项目是一个面向电商场景的商品库存与秒杀系统课程作业，目标是完成一个具备基础业务能力和一定高并发设计思想的后端系统原型。系统围绕用户、商品、库存、订单四个核心业务对象展开，重点体现秒杀业务下的库存控制、下单流程和系统拆分思路。

## 系统功能概述

系统计划包含以下核心模块：

### 1. 用户模块
负责用户注册、登录、用户信息查询等功能。

### 2. 商品模块
负责商品新增、修改、上下架、详情查询、列表展示等功能。

### 3. 库存模块
负责库存查询、库存扣减、库存回滚、库存锁定等功能，是秒杀业务的核心支撑模块。

### 4. 订单模块
负责普通下单、秒杀下单、订单查询、订单取消等功能。
